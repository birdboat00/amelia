amelia guide

The amelia guide walks you through setting up and using amelia for your generative-art projects. All examples in this guide are editable in the amelia web editor. Just click on the Look at code link on the bottom right on examples and it opens the code in the web editor.

If you are looking for the code documentation you can find it here.
